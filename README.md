# Grids

A browser-based tool for generating paper grid backgrounds. Choose your aspect ratio, grid style, density, and paper color, then export as a high-resolution JPEG or copy as SVG.

## Features

- **6 grid styles** — Line, Gap, Crosshair, Dot, Square, Diamond
- **4 aspect ratios** — 16:9, 1:1, 4:5, 9:16
- **Adjustable density** — 20 levels
- **10 paper colors** — warm parchment, aged cream, linen, and more
- **Export as JPEG** — full resolution (up to 1920×1080)
- **Copy as SVG** — scalable vector output

## Usage

Open `index.html` in any modern browser. No build step or dependencies required.
